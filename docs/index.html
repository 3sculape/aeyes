Having accomplished an OCR project last semester, the AEyes group is back with two new members for another challenge. This time, AEyes will tackle image manipulation. The goal is to create a functioning image editing software that will allow users to modify pictures in many ways. The project will revolve around many different algorithms that will be available to the end user on a graphical user interface.

February 2021

This marks the very beginning of the project AEyes. In the head of each team member there is one clear ambition: to create from scratch an innovative and powerful image processing software. The specifications are written, we are ready to start.

April 2021

After a selection of the technologies used and a fine tuning of the specifications of our software and a month of work on the project, we are able to deliver a first viable version of the AEyes software.

May 2021

During this second period of work, the watchword was to add a maximum of functionalities to the software in order to make it complete and functional. At the end of May 2021 we are able to provide a second version of AEyes Studio.

June 2021

During this last period of work the AEyes group gave its best to complete the software with more sophisticated functions as well as to eliminate all the bugs that would mar the user experience. We are proud to present you the final and most advanced version of AEyes Studio.
